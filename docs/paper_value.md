# 3rd Paper の価値 — North Star

> 細部に潜って「で、この論文の価値なんだっけ？」となったら、ここに戻る。
> （`/paper-value` skill と同内容のgit管理コピー。skill実体は `research/.claude/skills/`＝git管理外のため、こちらを正本バックアップとする。）

## 一文で言うと

**「薬剤が、がん細胞表面の糖鎖エピトープの"標的性・検出性"をどう書き換えるか」を、
LINCS薬剤応答から全薬剤×全エピトープで初めて定量し、HCCの糖鎖標的療法に橋渡しする。**

治す薬を探すのではない。**「腫瘍を抗体・レクチンで狙いやすく／見つけやすく変える薬」を見つける**、という一段手前の・でも創薬に直結する主張。

## 埋める空白（なぜ今までに無いか）

- 糖鎖エピトープは **抗体・レクチン・Siglec・セレクチン・galectin が実際に触る細胞表面の接点**。ここが変われば「検出できるか・狙えるか・がんの振る舞い」が全部変わる。
- 先行研究（GlycoMaple, Kawano 2005 等）は **静的な発現→糖鎖構造の予測**まで。
- **「薬剤で誘導される変化」を、エピトープ量として全薬剤網羅で定量し、標的性・創薬に接続した仕事は無い。** ここが新規性。

## 創薬・薬理への4つの出口（価値の本体）

```
薬 → glycogene発現変化 → glyco-epitope potential 変化
                          ├─ 検出できる → バイオマーカー/PD/コンパニオン診断（例 AFP-L3=core fucose を AAL で）
                          ├─ 狙える    → 抗体/ADC/CAR-T との併用・感作（標的密度を上げてから叩く）
                          ├─ MoA濃縮   → エピトープを動かす薬に共通する薬効群＝新規glyco制御標的の発見
                          └─ 振る舞い変化 → STn/sialyl Lewis/poly-LacNAc を下げる＝免疫回避・転移の抑制根拠
```

**一番刺さる出口**：出口2（併用・感作）＋出口1（AFP-L3的PDバイオマーカー）。HCCの **core fucose (FUT8/AFP-L3)** 軸がその具体例。

## この研究だからできること

1. **薬剤誘導**（静的予測でなく「薬でどう動くか」）
2. **全薬剤網羅**（LINCS）→ リポジショニング・MoA濃縮が言える
3. **エピトープ単位のスコア**（遺伝子でなく標的性の単位）→ 抗体/レクチン文脈に直結
4. **HCC文脈**（core fucose/AFP-L3, STn, Lewis/selectin, galectin）→ wet validation（HepG2 + AALレクチン）まで最短

## 正直な線引き（過剰主張しない）

- **主張する**：薬が糖鎖インターフェース（標的性・検出性）を再配線しうる。
- **主張しない**：薬が腫瘍を治す（直接の治療効果）。
- 既知の限界：**発現≠エピトープ量**（酵素の局在GALA・翻訳後制御で乖離）。transcriptomeシグナルは弱く、コントラスト対比や経路構造（step内max×step間min, 分岐競合）で分離を作る必要がある。
  → だからこそ「素朴な和でなく、生合成経路を反映したスコア」に価値がある。

## 迷ったときの判定基準

新しい解析・図を足すか迷ったら、**「これは4つの出口のどれを強めるか？」**で判断する。
どの出口にも効かないなら、それは論文の価値には効いていない。
